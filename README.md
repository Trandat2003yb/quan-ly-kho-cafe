# ☕ Coffee Warehouse Management System

## 📌 Overview
This is a warehouse management system for a coffee shop, developed using C# WinForms.

## 🛠 Technologies
- C# WinForms
- SQL Server
- 3-tier architecture (DAL, BLL, DTO)

## 🚀 Features
- Login / Register
- Manage products
- Import / Export warehouse
- Supplier management
- Inventory tracking
- Reports
